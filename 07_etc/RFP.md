## RFP

1. #### RFP?

   - RFP(Request for Proposal, 제안요청서)
   - 프로잭트를 담당할 업체를 선정하기 전에, 요구사항을 정리하여 후보 업체들에게 전달하는 문서

2. #### RFP 순서

   - 프로잭트 기획 >> 제안 요청서 작성 >> 요청서 심사 >> 제안서 작성 >>

     제안서 심사 >> 프로잭트 시작 및 수행 >> 프로잭트 발표
   
3. #### RFP 주요 내용

   1. 목표 설정
   2. 요구사항 명세
   3. 구현 요소 정리
   4. 제약 조건 도출
   5. 일정 수립(가능한)

   - 가급적 단순화 시켜서 서술

4. #### 지향 요소

   1. 명확성 / 단순성
   2. 가능성
   3. 자원 적극 활용

5. #### 지양 요소

   1. 불명확
   2. 다양한 기능 탑재
   3. 폐쇄적 기획
